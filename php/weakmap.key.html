WeakMap::key

(PECL weakref >= 0.2.0)

WeakMap::key — Returns the current key under iteration

说明

public WeakMap::key ( void ) : object

Returns the object serving as key in the map, at the current iterating position.

参数

此函数没有参数。

返回值

The object key currently being iterated.
