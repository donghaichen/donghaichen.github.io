The ReflectionType class

(PHP 7)

简介

ReflectionType 类用于获取函数、类方法的参数或者返回值的类型。

类摘要

ReflectionType {
/* 方法 */
public allowsNull ( void ) : bool
public isBuiltin ( void ) : bool
public __toString ( void ) : string
}
