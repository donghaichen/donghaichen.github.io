PDF_add_nameddest

(PECL pdflib >= 2.0.0)

PDF_add_nameddest — Create named destination

说明

PDF_add_nameddest ( resource $pdfdoc , string $name , string $optlist ) : bool

Creates a named destination on an arbitrary page in the current document. 成功时返回 TRUE， 或者在失败时返回 FALSE。
