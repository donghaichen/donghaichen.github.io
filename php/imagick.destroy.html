Imagick::destroy

(PECL imagick 2.0.0)

Imagick::destroy — Destroys the Imagick object

说明

Imagick::destroy ( void ) : bool

Destroys the Imagick object and frees all resources associated with it. This method is deprecated in favour of Imagick::clear.

返回值

成功时返回 TRUE。
