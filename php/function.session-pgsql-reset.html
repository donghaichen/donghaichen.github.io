(PECL session_pgsql SVN)

session_pgsql_reset — Reset connection to session database servers

说明

session_pgsql_reset ( void ) : bool

Reset the connection to the session database servers.

返回值

成功时返回 TRUE， 或者在失败时返回 FALSE。
