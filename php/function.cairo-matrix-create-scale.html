(PECL cairo >= 0.1.0)

cairo_matrix_create_scale — 别名 CairoMatrix::initScale()

说明

此函数是该函数的别名： CairoMatrix::initScale().

此函数别名已废弃，仅为了向后兼容而保留。不建议使用此函数，因为将来会从 PHP 中移除。
