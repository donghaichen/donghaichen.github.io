PDF_fit_pdi_page

(PECL pdflib >= 2.0.0)

PDF_fit_pdi_page — Place imported PDF page

说明

PDF_fit_pdi_page ( resource $pdfdoc , int $page , float $x , float $y , string $optlist ) : bool

Places an imported PDF page on the page, subject to various options. 成功时返回 TRUE， 或者在失败时返回 FALSE。
