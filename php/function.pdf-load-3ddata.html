PDF_load_3ddata

(PECL pdflib >= 2.1.0)

PDF_load_3ddata — Load 3D model

说明

PDF_load_3ddata ( resource $pdfdoc , string $filename , string $optlist ) : int

Loads a 3D model from a disk-based or virtual file.

This function requires PDF 1.6.
