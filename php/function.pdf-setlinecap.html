PDF_setlinecap

(PHP 4, PECL pdflib >= 1.0.0)

PDF_setlinecap — Set linecap parameter

说明

PDF_setlinecap ( resource $p , int $linecap ) : bool

Sets the linecap parameter to control the shape at the end of a path with respect to stroking.
